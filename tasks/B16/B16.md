## State-of-the-Art Cybersecurity Solutions

### Description
I conducted a survey of current state-of-the-art cybersecurity solutions used in modern organisations. The focus was on understanding how companies use advanced technologies such as AI-driven threat detection, Zero Trust architecture, cloud security tools, and endpoint protection systems to defend against cyber threats.

### Findings
1. AI-Driven Security (SIEM/XDR)

Modern systems such as Microsoft Sentinel and CrowdStrike Falcon use artificial intelligence to detect unusual behaviour across networks and devices in real time. These systems help identify threats faster than traditional manual monitoring.

2. Endpoint Detection and Response (EDR)

Tools like CrowdStrike Falcon continuously monitor endpoints (laptops, servers, devices) for suspicious activity such as malware execution or unusual processes, allowing rapid isolation of threats.

3. Zero Trust Architecture

Zero Trust security assumes that no user or device is trusted by default. Continuous authentication and least-privilege access are enforced, reducing the risk of internal and external breaches.

4. Cloud Security Posture Management (CSPM)

Cloud platforms such as AWS and Azure use security tools to detect misconfigurations that could expose sensitive data. This is important as organisations increasingly rely on cloud infrastructure.

5. Identity and Access Management (IAM)

Systems such as Okta manage user authentication and ensure users only access the systems they are authorised to use, improving overall security control.
